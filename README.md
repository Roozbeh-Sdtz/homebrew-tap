# Roozbeh-Sdtz Tap

## How do I install these formulae?

`brew install roozbeh-sdtz/tap/<formula>`

Or `brew tap roozbeh-sdtz/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "roozbeh-sdtz/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
